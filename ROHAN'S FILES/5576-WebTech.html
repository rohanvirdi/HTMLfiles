  • Unit 1

    Web Terminology:
    WebServer
    A web server is a system, either hardware or software that hosts and delivers content like web pages, images and videos to users over the internet.
    WebClient/ Browser
    A web client/browser is an application or tool that communicates with a web browser using Hypertext Transfer Protocol(HTTP)
    Understanding how a Browser communicates with a WebServer.
    Internet
    The network formed by the co-operative interconnection of millions of computers, linked together is called Internet
    Extranet
    Extranet is a type of network that allows users from the outside to access the intranet of an organization.
    Intranet
    An internal private network built within an organization using Internet and World Wide Standards and products that allows employees of an organization to gain access of the corporate information.
    WWW
    WWW comprises all the Web Pages, videos, pictures and other online content that can be accessed via a Web Browser.
    URL
    A Uniform Resource Locator (URL) is a reference to the address of the respurces on the network (internet) and a mechanism for retrieving it.
